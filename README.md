# Welcome to JavaHutt's personal webpack bundler

## Contains:
 - `pug`
 - `sass/scss`
 - `babel`
 - `sass/scss`
 - `eslint`
 - `pre-commit hooks`

## Usage:
 - `npm start or npm run start` - assemble dev build and launch dev server at http://localhost:8080/
 - `npm run build` - assemble production build
